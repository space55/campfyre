# Tools

To get started building your own Campfyre, you'll need the following tools.

1. 3D Printer
1. Soldering iron
1. Power drill
1. Wire cutter
1. Screwdriver set
1. Ring terminal crimp tool
1. Lighter/heat source (for heat shrink tubing)
1. 120-160 grit sand paper
1. Scissors
